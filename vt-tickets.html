- when you see a new VT/TGT ticket, before saying or doing anything check if the applicant has a note or a warning. If you see any, report to Support Team Managers

- If all seems good, proceed by sending the intro message below.

- NOTE: For Verified Partner, use "vp-intro". For Trusted Carry Partner, use "tcp-intro".


- after that, immedietly check the Rep of the applican by using /rep command.

- if they do not have the required amount of Rep, send them the NOT ENOUGH REP messege, of the respective type.


- Once the applicant has sent over all the Proof, you can go through and verify if all of them are $ or 5 star trades

- After that check the Rep Log of the applicant in #|investigations channel, by using /replog command.

- Make sure the Trades have been repped on the server, and check if there was any rep abuse, a few missing Rep on the trades is acceptable.


- Send the closing text and make sure they agree with the terms.

- NOTE: For Verified Partner, use "vp-close". For Trusted Carry Partner, use "tcp-close".


- You can go ahead and close the ticket if they agree, and don't have any more questions
